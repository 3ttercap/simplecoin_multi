Building a strong and inexpensive rig case

This guide covers instructions on how to build a chassis for holding mining computers using many graphics cards. While it may not be too pretty, it's easy to build, and fairly forgiving with measurements and mistakes. Our first version (pictured on the right) was done without using any measuring tools (just the parts for mockups) and this guide documents how we built the second version.
